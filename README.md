# encodepad
=====

Encodepad 是python GUI加解密测试工具箱，用来测试各种文本或二进制内容的加解密，支持md5/sha/rsa/rc4/des/aes等加密算法。

## 安装依赖库
    pip install wxpython
	pip install pycrypto
  
## Run
	双击运行，如讨厌出现的dos窗口，可重命名后缀*.py至*.pyw。
